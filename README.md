# open-nbj-tools

Free tools built for other that appreciate it. 

## Development

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Scripts

- `pnpm lint` / `pnpm lint:fix`
- `pnpm format`
- `pnpm build` / `pnpm start`
- `pnpm translations` (extract/push/pull/compile)
